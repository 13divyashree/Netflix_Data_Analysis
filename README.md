# 🎬 Netflix Data Analysis Dashboard

## Description
A data visualization dashboard analyzing the Netflix dataset to explore content trends, genres, and distribution.

## Tools Used
- Power BI
- Data Visualization
- Data Analysis

## Key Insights
- Movies dominate Netflix content compared to TV shows
- Certain genres are more popular across the platform
- Content addition has increased over the years
